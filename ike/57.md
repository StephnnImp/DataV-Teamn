## Parameters

Fixed length. 1-byte bitfield.

    STATE   = 0b0110_0000
    BUTTON  = 0b0000_0011

### State `0b0110_000`

    STATE_PRESS     = 0b00
    STATE_RELEASE   = 0b10

No hold event!

### Button `0b0000_0011`

    BUTTON_CHECK    = 0b01  # CHECK button on high clusters
    BUTTON_BC       = 0b10  # BC button on stalk

I'm only aware of the two buttons.
